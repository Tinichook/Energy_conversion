# 风机功率曲线工具快速指南

## 📁 工具文件

### 🌟 推荐使用：多曲线对比版本（v2.1）
**文件**: `wind-power-multi-curve.html`

**一键打开**: 双击文件，用浏览器打开即可

**核心功能**:
- ✅ 静态功率曲线多风机对比（P-V）
- ✅ Cp曲线多风机对比（Cp-V）
- ✅ **图例显示在图表内部** ⭐新特性
- ✅ **日功率曲线支持多风机对比** ⭐新特性
- ✅ 每个风机独立显示统计数据卡片
- ✅ 复选框灵活选择风机组合

**使用步骤**:
1. 双击打开 `wind-power-multi-curve.html`
2. **静态曲线区域**：
   - 勾选要对比的风机型号（可多选）
   - 点击"🔄 更新静态曲线"按钮
   - 查看左右两张对比图（图例在图内左上角）
3. **日功率区域**：
   - 勾选要对比的风机型号（可多选）⭐
   - 选择季节（冬/春/夏/秋）
   - 点击"🔄 更新日功率曲线"按钮
   - 查看多条功率曲线和各风机统计数据 ⭐

## 💡 使用技巧

### 静态曲线对比
- **同功率等级对比**: 选择2-3个相近功率的风机
  - 例如：SWT-3kW + SWT-10kW + MWT-50kW
- **不同功率等级对比**: 选择小中大各一个
  - 例如：SWT-10kW + MWT-50kW + GW121/2500
- **全局对比**: 全选7个风机，看整体趋势

### 日功率分析 ⭐多风机对比
- **同季节多风机对比**: 
  - 选择2-3个风机
  - 固定季节（如夏季）
  - 对比不同风机的发电能力
  
- **同风机不同季节对比**:
  - 选择1个风机
  - 切换不同季节
  - 分析季节性差异

- **多风机多季节综合分析**:
  - 选择多个风机
  - 记录不同季节的数据
  - 进行全面对比

### 发电量估算
- **年发电量估算公式**:
  ```
  年发电量 ≈ 日发电量 × 365 × 容量因子
  容量因子通常为 0.25-0.35
  ```

- **示例**（GW121/2500）:
  - 夏季日发电量：35,000 kWh
  - 冬季日发电量：50,000 kWh
  - 平均日发电量：42,500 kWh
  - 年发电量估算：42,500 × 365 × 0.3 ≈ 4,653,750 kWh ≈ 4.65 GWh

### 选型参考
- **小型应用**: 选择 SWT-3kW 或 SWT-10kW
  - 日发电量：30-200 kWh
  - 适合家庭、农场
  
- **中型应用**: 选择 MWT-50kW
  - 日发电量：500-1,200 kWh
  - 适合小型企业
  
- **大型风场**: 选择 1500-3600 kW 的大型风机
  - 日发电量：30,000-55,000 kWh
  - 适合风电场

## 🔧 技术参数

### 计算公式
```
P = (1/8) × Cp × η₁ × η₂ × η₃ × ρ × π × D² × V³
```

### 效率参数
- η₁ = 0.92 (传动效率)
- η₂ = 0.95 (发电机效率)
- η₃ = 0.95 (变流器效率)
- ρ = 1.225 kg/m³ (空气密度)

### 数据来源
- 基于表A.2各风速下不同风机Cp表
- 风速步长：0.5 m/s
- 风速范围：0-25 m/s

## ❓ 常见问题

**Q: 打不开网页？**
- A: 确保使用Chrome、Edge、Firefox等现代浏览器

**Q: 图表不显示？**
- A: 检查是否至少选择了一个风机，点击"更新"按钮

**Q: 图例在哪里？** ⭐
- A: 图例在每个图表的内部左上角，半透明白色背景

**Q: 日功率图可以对比多个风机吗？** ⭐
- A: 可以！勾选多个风机，每个风机会显示不同颜色的曲线和独立的统计卡片

**Q: 如何对比不同风机的发电能力？** ⭐
- A: 
  1. 在日功率区域勾选要对比的风机（2-3个为佳）
  2. 选择同一季节
  3. 点击更新按钮
  4. 查看不同颜色的曲线和统计卡片
  5. 对比日发电量数据

**Q: 统计卡片的颜色有什么含义？** ⭐
- A: 每个卡片的边框和数据颜色与对应曲线的颜色一致，方便识别

**Q: 如何估算年发电量？**
- A: 年发电量 ≈ 日发电量 × 365 × 容量因子（通常0.25-0.35）

## 🎯 典型应用场景

### 场景1：风机选型对比
1. 在静态曲线区域选择候选风机（如GW121/2500、MySE3.0-135）
2. 对比功率曲线和Cp曲线
3. 在日功率区域选择相同风机
4. 对比日发电量
5. 综合考虑性能和发电能力做出选型决策

### 场景2：季节性发电分析
1. 在日功率区域选择一个风机
2. 依次选择不同季节（冬/春/夏/秋）
3. 记录每个季节的日发电量
4. 分析季节性差异
5. 估算年发电量

### 场景3：多风机组合方案
1. 在日功率区域选择多个不同功率等级的风机
2. 查看各风机的日发电量
3. 计算多台风机组合的总发电量
4. 优化风机组合方案
