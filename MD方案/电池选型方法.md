# 储能电池系统选型方法

## 一、储能系统设计原则

### 1.1 储能系统的作用

在风光互补发电系统中，储能系统承担以下关键功能：

1. **功率平滑**：平抑风光发电的短期波动（秒级-分钟级）
2. **削峰填谷**：在发电高峰时储能，用电高峰时放电（小时级）
3. **备用电源**：在风光出力不足时提供应急电力
4. **电能质量调节**：改善电压、频率稳定性

### 1.2 储能容量配置原则

储能系统容量需要综合考虑：
- 可再生能源装机容量
- 负荷特性（峰谷差、波动性）
- 自给率目标
- 经济性约束

## 二、储能容量计算方法

### 2.1 基于功率平滑的容量计算

**目标**：平抑风光发电的短期波动


$$E_{smooth} = P_{RE} \times \Delta t \times k_1 \qquad (1)$$

式中：
- $E_{smooth}$ —— 功率平滑所需储能容量，kWh
- $P_{RE}$ —— 可再生能源装机容量，kW
- $\Delta t$ —— 平滑时间窗口，一般取0.25～0.5小时
- $k_1$ —— 波动系数，风电取0.2～0.3，光伏取0.15～0.25

**示例**：某区域风电装机12 MW，光伏装机18 MW
- 风电平滑容量：$E_{wind} = 12000 \times 0.25 \times 0.25 = 750$ kWh
- 光伏平滑容量：$E_{solar} = 18000 \times 0.25 \times 0.2 = 900$ kWh
- 合计：$E_{smooth} = 1650$ kWh

### 2.2 基于削峰填谷的容量计算

**目标**：在发电与负荷不匹配时进行能量转移

$$E_{shift} = \int_{t_1}^{t_2} |P_{gen}(t) - P_{load}(t)| dt \qquad (2)$$

简化计算方法：

$$E_{shift} = (P_{load,peak} - P_{gen,valley}) \times T_{peak} \times k_2 \qquad (3)$$

式中：
- $E_{shift}$ —— 削峰填谷所需储能容量，kWh
- $P_{load,peak}$ —— 负荷峰值功率，kW
- $P_{gen,valley}$ —— 发电谷值功率，kW
- $T_{peak}$ —— 峰荷持续时间，小时
- $k_2$ —— 储能参与系数，取0.3～0.5

**示例**：居民区峰值负荷50 MW，夜间风光发电仅10 MW，峰荷持续3小时
$$E_{shift} = (50000 - 10000) \times 3 \times 0.4 = 48000 \text{ kWh}$$


### 2.3 基于自给率目标的容量计算

**目标**：提高系统自给率，减少对外部电网依赖

$$E_{target} = E_{daily,load} \times (1 - \alpha) \times \beta \qquad (4)$$

式中：
- $E_{target}$ —— 目标储能容量，kWh
- $E_{daily,load}$ —— 日用电量，kWh
- $\alpha$ —— 当前自给率
- $\beta$ —— 储能贡献系数，取0.15～0.25

**示例**：居民区日用电660 MWh，当前自给率28%，目标提升至40%
$$E_{target} = 660000 \times (1 - 0.28) \times 0.2 = 95040 \text{ kWh}$$

### 2.4 综合容量确定

取以上三种方法计算结果的加权平均：

$$E_{total} = w_1 E_{smooth} + w_2 E_{shift} + w_3 E_{target} \qquad (5)$$

权重建议：
- 工业区/居民区（负荷稳定）：$w_1=0.2, w_2=0.5, w_3=0.3$
- 山地区/农业区（发电为主）：$w_1=0.3, w_2=0.4, w_3=0.3$
- 林业区（小规模）：$w_1=0.4, w_2=0.4, w_3=0.2$

## 三、储能功率计算方法

### 3.1 充电功率计算

充电功率应满足在发电高峰时段快速吸收多余电能：

$$P_{charge} = \frac{E_{total}}{T_{charge}} \times \frac{1}{\eta_{charge}} \qquad (6)$$

式中：
- $P_{charge}$ —— 充电功率，kW
- $T_{charge}$ —— 充电时长，一般取2～4小时
- $\eta_{charge}$ —— 充电效率，磷酸铁锂取0.95，铅碳取0.85


### 3.2 放电功率计算

放电功率应满足在用电高峰时段支撑负荷：

$$P_{discharge} = P_{load,peak} \times k_3 \qquad (7)$$

式中：
- $P_{discharge}$ —— 放电功率，kW
- $k_3$ —— 储能支撑系数，取0.15～0.25

**或者**根据放电时长计算：

$$P_{discharge} = \frac{E_{total} \times DOD}{T_{discharge}} \times \eta_{discharge} \qquad (8)$$

式中：
- $DOD$ —— 放电深度，磷酸铁锂取0.9，铅碳取0.7
- $T_{discharge}$ —— 放电时长，一般取3～5小时
- $\eta_{discharge}$ —— 放电效率，磷酸铁锂取0.95，铅碳取0.85

### 3.3 储能功率确定

$$P_{ESS} = \max(P_{charge}, P_{discharge}) \qquad (9)$$

一般情况下，充放电功率取相近值，按较大值配置。

## 四、电池类型选择

### 4.1 电池技术对比

| 参数 | 磷酸铁锂 | 三元锂 | 铅碳电池 | 液流电池 |
|------|---------|--------|---------|---------|
| 能量密度(Wh/kg) | 120-160 | 180-250 | 35-50 | 15-25 |
| 循环寿命(次) | 6000-8000 | 3000-5000 | 3000-4000 | 10000+ |
| 放电深度(%) | 90 | 80 | 70 | 100 |
| 充放电效率(%) | 95 | 93 | 85 | 75 |
| 工作温度(°C) | -20～55 | -10～45 | -10～45 | 5～40 |
| 安全性 | 高 | 中 | 高 | 极高 |
| 单位成本(元/kWh) | 2200-2600 | 2500-3000 | 1200-1500 | 3000-4000 |
| 全生命周期成本 | 低 | 中 | 中 | 低 |


### 4.2 电池选型决策树

```
                        储能系统选型
                            |
            ┌───────────────┴───────────────┐
            |                               |
        日循环次数                       应用场景
            |                               |
    ┌───────┴───────┐               ┌───────┴───────┐
    |               |               |               |
  >1次/天        <1次/天         固定式          移动式
    |               |               |               |
磷酸铁锂/三元锂    铅碳电池      磷酸铁锂        三元锂
```

**选型建议**：

1. **磷酸铁锂电池**（推荐）
   - 适用场景：日循环1次以上，需要长寿命
   - 优点：循环寿命长、安全性高、维护少
   - 缺点：初始投资较高
   - 推荐区域：工业区、居民区、农业区

2. **铅碳电池**
   - 适用场景：日循环<1次，预算有限
   - 优点：初始成本低、技术成熟
   - 缺点：寿命较短、效率较低、需定期维护
   - 推荐区域：山地区、林业区（小规模）

3. **三元锂电池**
   - 适用场景：空间受限、需要高能量密度
   - 优点：能量密度高、体积小
   - 缺点：安全性相对较低、成本高
   - 推荐区域：特殊应用场景

## 五、电池组串并联设计

### 5.1 电池组电压设计

储能系统电压等级选择：

| 系统规模 | 推荐电压等级 | 说明 |
|---------|-------------|------|
| <100 kWh | 48V | 小型系统，安全性高 |
| 100-500 kWh | 384V-768V | 中型系统，效率较高 |
| 500-2000 kWh | 768V-1500V | 大型系统，减少电流损耗 |
| >2000 kWh | 1000V-1500V | 超大型系统，集装箱式 |


### 5.2 串并联计算

**串联数量**：

$$N_s = \frac{V_{system}}{V_{cell}} \qquad (10)$$

式中：
- $N_s$ —— 串联电池单体数量
- $V_{system}$ —— 系统电压，V
- $V_{cell}$ —— 单体电池电压，磷酸铁锂为3.2V

**并联数量**：

$$N_p = \frac{E_{total}}{V_{system} \times C_{cell}} \qquad (11)$$

式中：
- $N_p$ —— 并联组数
- $C_{cell}$ —— 单体电池容量，Ah

**示例**：设计5000 kWh / 768V 储能系统，采用280Ah磷酸铁锂电池

1. 串联数：$N_s = 768 / 3.2 = 240$ 节（组成768V电池组）
2. 单组容量：$E_{pack} = 768 \times 280 / 1000 = 215$ kWh
3. 并联数：$N_p = 5000 / 215 = 23.3 \approx 24$ 组
4. 实际容量：$E_{actual} = 215 \times 24 = 5160$ kWh

### 5.3 电池组配置优化

**配置原则**：
1. 串联数量应考虑PCS电压范围（一般600-1500V）
2. 并联数量不宜过多（建议≤30组），避免环流问题
3. 预留10-15%容量裕度，补偿衰减
4. 考虑标准化模块（如48V/51.2V电池簇）

## 六、储能变流器(PCS)选型

### 6.1 PCS功率配置

PCS额定功率应满足：

$$P_{PCS} \geq P_{ESS} \times 1.1 \qquad (12)$$

预留10%裕度，应对峰值功率需求。

### 6.2 PCS电压匹配

PCS电池侧电压范围应覆盖电池组工作电压：

$$V_{batt,min} \leq V_{PCS,min} < V_{system} < V_{PCS,max} \leq V_{batt,max} \qquad (13)$$


### 6.3 PCS数量配置

**单台PCS方案**：
- 适用：<500 kW系统
- 优点：结构简单、成本低
- 缺点：单点故障风险

**多台PCS并联方案**：
- 适用：>500 kW系统
- 优点：冗余度高、可靠性好、便于扩容
- 缺点：控制复杂、成本较高

**配置建议**：

$$N_{PCS} = \lceil \frac{P_{ESS}}{P_{PCS,unit}} \rceil \qquad (14)$$

一般单台PCS功率选择100-500 kW。

## 七、电池管理系统(BMS)设计

### 7.1 BMS功能要求

**基本功能**：
1. 电压监测（单体/模组/系统）
2. 电流监测（充放电电流）
3. 温度监测（多点温度传感器）
4. SOC估算（库仑计量法+开路电压法）
5. SOH评估（容量衰减、内阻增长）

**保护功能**：
1. 过充保护（单体电压>3.65V）
2. 过放保护（单体电压<2.5V）
3. 过流保护（充放电电流超限）
4. 过温保护（温度>55°C）
5. 短路保护
6. 绝缘检测

**均衡功能**：
1. 被动均衡：通过电阻放电，简单但效率低
2. 主动均衡：能量转移，效率高但成本高
3. 均衡电流：一般50-200mA

### 7.2 BMS架构选择

| 架构类型 | 集中式 | 分布式 | 主从式 |
|---------|--------|--------|--------|
| 适用规模 | <100 kWh | 任意 | >500 kWh |
| 可靠性 | 低 | 高 | 中 |
| 扩展性 | 差 | 好 | 好 |
| 成本 | 低 | 高 | 中 |
| 推荐场景 | 小型系统 | 分布式储能 | 大中型系统 |


## 八、热管理系统设计

### 8.1 热管理必要性

磷酸铁锂电池最佳工作温度：15-35°C
- 温度过高（>45°C）：加速衰减、安全风险
- 温度过低（<0°C）：容量下降、内阻增大
- 温差过大（>5°C）：电池不一致性加剧

### 8.2 热管理方式

**自然冷却**：
- 适用：<100 kWh，环境温度适宜
- 方式：通风散热
- 成本：低

**强制风冷**：
- 适用：100-1000 kWh
- 方式：风扇+风道设计
- 优点：成本适中、维护简单
- 缺点：温度均匀性一般

**液冷系统**：
- 适用：>1000 kWh，高倍率充放电
- 方式：冷却液循环（水/乙二醇）
- 优点：温控精确、温度均匀
- 缺点：成本高、维护复杂

### 8.3 热管理功率估算

$$P_{cooling} = P_{ESS} \times (1 - \eta_{ESS}) \times k_4 \qquad (15)$$

式中：
- $P_{cooling}$ —— 散热功率，kW
- $k_4$ —— 散热系数，风冷取1.2，液冷取1.5

## 九、储能系统经济性分析

### 9.1 全生命周期成本(LCC)

$$LCC = C_{initial} + C_{operation} + C_{replacement} - C_{residual} \qquad (16)$$

**初始投资**：
$$C_{initial} = C_{battery} + C_{PCS} + C_{BMS} + C_{thermal} + C_{installation} \qquad (17)$$


**运维成本**：
$$C_{operation} = \sum_{i=1}^{n} \frac{C_{O\&M,i}}{(1+r)^i} \qquad (18)$$

式中：
- $C_{O\&M,i}$ —— 第i年运维成本
- $r$ —— 贴现率，一般取5-8%
- $n$ —— 系统寿命，年

**更换成本**：
- 磷酸铁锂：10年后可能需要更换部分电池模组
- 铅碳电池：5-7年需要整体更换

### 9.2 度电成本(LCOS)

$$LCOS = \frac{LCC}{\sum_{i=1}^{n} \frac{E_{throughput,i}}{(1+r)^i}} \qquad (19)$$

式中：
- $E_{throughput,i}$ —— 第i年累计充放电量，kWh

**示例**：5000 kWh磷酸铁锂储能系统
- 初始投资：1700万元（含电池、PCS、BMS、安装）
- 日循环1次，年充放电量：5000 × 365 × 0.9 = 1642500 kWh
- 10年寿命，循环3650次，总吞吐量：16425000 kWh
- LCOS ≈ 1700 / 1642.5 ≈ 1.04 元/kWh

### 9.3 投资回收期

$$T_{payback} = \frac{C_{initial}}{R_{annual} - C_{O\&M,annual}} \qquad (20)$$

式中：
- $R_{annual}$ —— 年收益（峰谷价差套利、辅助服务等）

## 十二、储能系统运行策略

### 12.1 充放电策略

**策略1：峰谷套利**
- 谷时（23:00-7:00）：充电，吸收低价电能
- 平时（7:00-10:00, 15:00-18:00）：待机
- 峰时（10:00-15:00, 18:00-23:00）：放电，支撑负荷


**策略2：跟踪可再生能源**
- 发电>负荷：充电，存储多余电能
- 发电<负荷：放电，补充电力缺口
- 发电≈负荷：待机

**策略3：削峰填谷**
- 负荷>阈值（如80% Pmax）：放电
- 负荷<阈值（如30% Pmax）且发电充足：充电
- 其他时段：待机

### 12.2 SOC管理

**SOC上下限设置**：
- SOC_max = 95%（避免过充）
- SOC_min = 10%（避免过放）
- SOC_target = 50%（目标SOC，平衡充放电能力）

**SOC分区控制**：
- SOC > 90%：限制充电功率至0.5C
- SOC < 20%：限制放电功率至0.5C
- 20% ≤ SOC ≤ 90%：正常充放电

### 12.3 温度管理策略

**温度分级控制**：
- T < 0°C：预热，限制充电至0.2C
- 0°C ≤ T < 15°C：正常运行，启动加热
- 15°C ≤ T ≤ 35°C：最佳工作区间
- 35°C < T ≤ 45°C：启动冷却系统
- T > 45°C：降功率运行，强制冷却
- T > 55°C：停机保护

## 十三、储能系统安全设计

### 13.1 电气安全

1. **绝缘监测**：实时监测正负极对地绝缘电阻，阈值>100kΩ
2. **漏电保护**：配置剩余电流保护器，动作电流≤300mA
3. **短路保护**：快速熔断器+断路器双重保护
4. **过流保护**：电流传感器实时监测，超限立即切断

### 13.2 消防安全

1. **烟雾探测**：每个电池柜配置烟雾探测器
2. **温度监测**：多点温度传感器，异常升温报警
3. **气体探测**：监测H2、CO等可燃气体浓度
4. **灭火系统**：
   - 小型系统：干粉/CO2灭火器
   - 大型系统：气溶胶/全氟己酮自动灭火


### 13.3 机械安全

1. **抗震设计**：按当地抗震烈度设计（一般7-8度）
2. **防护等级**：电池柜IP54以上，户外IP65
3. **通风设计**：自然通风+强制通风，换气次数≥6次/h
4. **防爆设计**：电气设备采用防爆型（Ex d IIB T4）

### 13.4 多级保护体系

        ┌─────────────────┼─────────────────┐

1. 一级报警：参数接近限值，记录日志
2. 二级报警：参数超限，降功率运行

4. 紧急停机：极端情况，断开所有连接

## 十五、储能系统性能评估指标

### 15.1 技术指标

| 指标 | 计算公式 | 目标值 |
|------|---------|--------|
| 充电效率 | $\eta_c = E_{stored} / E_{input}$ | >95% |
| 放电效率 | $\eta_d = E_{output} / E_{stored}$ | >95% |
| 往返效率 | $\eta_{RT} = \eta_c \times \eta_d$ | >90% |
| 可用率 | $A = T_{available} / T_{total}$ | >98% |
| 响应时间 | 从指令到输出 | <100ms |

### 15.2 经济指标

| 指标 | 计算公式 | 说明 |
|------|---------|------|
| 度电成本 | $LCOS = LCC / E_{lifetime}$ | 全生命周期度电成本 |
| 投资回收期 | $T = C_{initial} / R_{annual}$ | 静态回收期 |
| 内部收益率 | IRR计算 | 项目盈利能力 |
| 净现值 | NPV计算 | 项目价值评估 |

### 15.3 环境指标

| 指标 | 计算公式 | 说明 |
|------|---------|------|
| 碳减排量 | $\Delta CO_2 = E_{ESS} \times EF$ | EF为电网排放因子 |
| 可再生能源利用率 | $\eta_{RE} = E_{RE,used} / E_{RE,total}$ | 提升可再生能源消纳 |

## 十七、常见问题与解答

**Q1：为什么不能简单地按"额定电压48V、容量1120Ah"来选电池？**

A：这种简化选型存在以下问题：
1. **电压不匹配**：48V系统功率太小，无法满足MW级储能需求
2. **容量不合理**：未考虑实际充放电功率、时长、DOD等因素
3. **缺乏系统性**：未考虑PCS匹配、BMS设计、热管理等
4. **经济性差**：未优化串并联配置，可能造成浪费

**Q2：磷酸铁锂和铅碳电池如何选择？**

A：决策依据：
- **日循环>1次** → 磷酸铁锂（寿命长，LCOS低）
- **日循环<1次** → 铅碳电池（初投资低）
- **预算充足** → 磷酸铁锂（长期更经济）
- **预算紧张** → 铅碳电池（短期成本低）


**Q3：储能容量是越大越好吗？**

A：不是。储能容量需要平衡：
- **容量过小**：无法有效平抑波动，自给率提升有限
- **容量过大**：初投资高，利用率低，LCOS升高
- **最优容量**：根据公式5综合计算，一般为日用电量的5-15%

**Q4：为什么要配置多台PCS而不是一台大功率PCS？**

A：多台PCS并联的优势：
1. **可靠性**：单台故障不影响整体运行
2. **灵活性**：可根据功率需求启停部分PCS
3. **扩展性**：便于后期扩容
4. **维护性**：可轮流检修，不停机

**Q5：BMS的均衡功能有多重要？**

A：非常重要。电池不一致性会导致：
- 容量损失：木桶效应，最弱电池限制整组容量
- 寿命缩短：弱电池过充过放，加速衰减
- 安全隐患：电压差过大可能引发热失控

主动均衡优于被动均衡，但成本更高。

**Q6：储能系统的实际寿命有多长？**

A：取决于多个因素：
- **循环寿命**：磷酸铁锂6000次，铅碳3000次
- **日历寿命**：10-15年
- **实际寿命**：取两者较小值

示例：日循环1次的磷酸铁锂系统
- 循环寿命：6000次 ÷ 365天 = 16.4年
- 日历寿命：10-15年
- **实际寿命：约10-12年**
