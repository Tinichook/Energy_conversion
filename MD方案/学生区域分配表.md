# 学生区域分配表

## 分配规则

学生登录后的区域分配基于**学号后4位取模算法**，对于冲突的区域，将重复学生分配到空闲区域。

**原算法：** `区域ID = (学号后4位 % 52) + 1`

**代码位置：** 
- 映射表：`energy-system/src/students.ts` 中的 `studentRegionMap`
- 登录逻辑：`energy-system/src/App.tsx` 中的 `handleLogin` 函数
