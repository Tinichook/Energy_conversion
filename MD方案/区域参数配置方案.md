# 能源系统区域参数配置方案

## 一、各区域特性总览

| 区域类型 | 风能资源 | 光能资源 | 生物质资源 | 电力消耗 | 建设成本 | 使用成本 |
|---------|---------|---------|-----------|---------|---------|---------|
| 工业区 | 正常 | 正常 | 很低(生活垃圾) | 最高(夜>日) | 最高 | 高 |
| 居民区 | 正常 | 正常 | 低(生活垃圾) | 高(日>夜) | 高 | 高 |
| 山地区 | 充足(+30%) | 充足(+20%) | 较低(灌木) | 低(日高夜低) | 正常 | 正常 |
| 农业区 | 充足(+10%) | 充足(+10%) | 高(季节性秸秆) | 中等 | 超量后高 | 低 |
| 林业区 | 匮乏(-40%) | 匮乏(-30%) | 最高(优质木屑) | 最低 | 高(从0开始) | 最低 |

### 1.1 各区域能源自给能力概览

| 区域类型 | 日用电(MWh) | 生物质发电 | 风能发电 | 太阳能发电 | 总发电(MWh) | 自给率 |
|---------|------------|-----------|---------|-----------|------------|-------|
| 工业区 | 1320 | 47 (4%) | 82 (6%) | 66 (5%) | 195 | 15% |
| 居民区 | 660 | 69 (10%) | 59 (9%) | 59 (9%) | 187 | 28% |
| 山地区 | 120 | 35 (29%) | 171 (143%) | 58 (48%) | 264 | 220% |
| 农业区 | 240 | 202 (84%) | 120 (50%) | 89 (37%) | 411 | 171% |
| 林业区 | 72 | 210 (292%) | 21 (29%) | 22 (31%) | 253 | 351% |

**设计理念：**
- 工业区/居民区：能源消耗大户，自给率低，需要从其他区域调入能源
- 山地区/农业区/林业区：能源生产区，自给率高，可向外输出
- 小组协作：通过生物质运输和电力传输实现资源互补

## 二、风能资源参数设计

### 2.1 基础参数（参考中国东部平原地区）

| 参数 | 说明 | 参考值 |
|-----|------|-------|
| 年平均风速 | 中国东部平原 | 3-5 m/s |
| 有效风速范围 | 风机启动-额定 | 3-25 m/s |
| 风能密度 | 年平均 | 100-300 W/m² |

### 2.2 各区域风能配置

```javascript
风能配置 = {
  工业区: {
    baseSpeed: 4.0,      // 基础风速 m/s
    variance: 3.0,       // 随机波动范围 (±1.5 m/s)
    dailyAmplitude: 1.5, // 日内波动幅度
    bonus: 0,            // 额外加成
    randomFactor: 0.8    // 随机性系数（0.8=较稳定）
  },
  居民区: {
    baseSpeed: 3.5,      // 城区风速略低
    variance: 2.5,
    dailyAmplitude: 1.5,
    bonus: 0,
    randomFactor: 0.7
  },
  山地区: {
    baseSpeed: 5.5,      // 山区风速高
    variance: 4.0,       // 波动大
    dailyAmplitude: 2.5,
    bonus: 2.0,          // 地形加成
    randomFactor: 1.0    // 随机性最强
  },
  农业区: {
    baseSpeed: 4.5,      // 开阔地带风速好
    variance: 3.5,
    dailyAmplitude: 2.0,
    bonus: 0.5,
    randomFactor: 0.9
  },
  林业区: {
    baseSpeed: 2.5,      // 林区风速低（树木阻挡）
    variance: 2.0,
    dailyAmplitude: 1.0,
    bonus: -1.0,         // 负加成
    randomFactor: 0.6    // 较稳定但低
  }
}
```

### 2.3 风能随机性模型

```
风速(t) = baseSpeed + bonus 
        + sin(t/3) × dailyAmplitude           // 日内周期变化
        + random() × variance × randomFactor   // 随机波动
        + 突发事件因子                          // 无风/大风天
```

**突发事件概率：**
- 无风天（风速<2m/s）：10%概率
- 大风天（风速>8m/s）：5%概率
- 持续刮风（6小时以上）：30%概率

### 3.2 日内光照强度曲线

```
光照强度(hour) = {
  hour < 日出 或 hour > 日落: 0
  否则: 峰值强度 × sin(π × (hour - 日出) / 日照时长) × 天气系数
}

峰值时段: 11:00 - 15:00 (太阳高度角最大)
峰值强度: 0.8 - 1.0 kW/m² (晴天)
```

### 3.3 各区域光能配置

```javascript
光能配置 = {
  工业区: {

    seasonalAmplitude: 0.3,   // 季节波动幅度
    variance: 0.15,           // 天气随机波动
    multiplier: 1.0,          // 区域系数
    cloudCover: 0.2           // 平均云量（城市热岛效应）
  },
  居民区: {
    baseIntensity: 0.5,
    seasonalAmplitude: 0.3,
    variance: 0.15,

    variance: 0.1,
    multiplier: 0.7,          // -30%
    cloudCover: 0.25
  }
```

## 四、生物质资源参数设计

### 4.1 生物质发电能力计算

**基础换算：**
- 生物质热值：10-18 MJ/kg（因类型而异）
- 发电效率：25-30%（取28%）
- 1吨生物质发电量（按热值计算）：
  - 工业区（10 MJ/kg）：10000 × 0.28 ÷ 3.6 ≈ **0.78 MWh**
  - 居民区（11 MJ/kg）：11000 × 0.28 ÷ 3.6 ≈ **0.86 MWh**
  - 山地区（15 MJ/kg）：15000 × 0.28 ÷ 3.6 ≈ **1.17 MWh**
  - 农业区（16 MJ/kg）：16000 × 0.28 ÷ 3.6 ≈ **1.24 MWh**
  - 林业区（18 MJ/kg）：18000 × 0.28 ÷ 3.6 ≈ **1.40 MWh**

### 4.2 各区域生物质产量（现实合理范围）

| 区域类型 | 生物质来源 | 日产量(吨) | 热值(MJ/kg) | 可发电量(MWh/日) | 占本区日用电比例 |
|---------|-----------|-----------|------------|-----------------|----------------|
| 工业区 | 生活垃圾/工业废料 | 50-80 | 10 | 39-62 | 3-5% |
| 居民区 | 生活垃圾 | 60-100 | 11 | 52-86 | 8-13% |
| 山地区 | 灌木枝条 | 20-40 | 15 | 23-47 | 19-39% |
| 农业区 | 秸秆(季节性) | 80-400 | 16 | 99-496 | 41-207%* |
| 林业区 | 木屑/树皮 | 120-180 | 18 | 168-252 | 233-350% |

*农业区丰收季节产量高，可供应其他区域；非丰收季产量低
*林业区全年稳定产出，是小组内主要的生物质供应来源

### 4.3 生物质产量详细配置

```javascript
生物质配置 = {
  工业区: {
    baseOutput: 60,       // 基础日产量（吨/天）
    harvestOutput: 60,    // 无季节性变化
    harvestMonths: [],    // 无丰收季
    variance: 15,         // 波动范围
    // 年产量：约 21900 吨，可发电 17082 MWh
    // 日均发电：46.8 MWh，占日用电1200MWh的 3.9%
  },
  居民区: {
    baseOutput: 80,       // 基础日产量（吨/天）
    harvestOutput: 80,
    harvestMonths: [],
    variance: 20,
    // 年产量：约 29200 吨，可发电 25112 MWh
    // 日均发电：68.8 MWh，占日用电660MWh的 10.4%
  },
  山地区: {
    baseOutput: 30,       // 基础日产量（吨/天）- 采集困难
    harvestOutput: 30,
    harvestMonths: [],
    variance: 10,
    // 年产量：约 10950 吨，可发电 12812 MWh
    // 日均发电：35.1 MWh，占日用电120MWh的 29.3%
  },
  农业区: {
    baseOutput: 100,      // 非丰收季日产量（吨/天）
    harvestOutput: 350,   // 丰收季日产量（吨/天）
    harvestMonths: [8, 9, 10],  // 8-10月丰收季（秋收）
    variance: 30,
    // 年产量：约 59400 吨，可发电 73656 MWh
    // 日均发电：201.8 MWh，占日用电240MWh的 84.1%
  },
  林业区: {
    baseOutput: 150,      // 基础日产量（吨/天）- 全年稳定，最高产量
    harvestOutput: 150,
    harvestMonths: [],
    variance: 20,
    // 年产量：约 54750 吨，可发电 76650 MWh
    // 日均发电：210 MWh，占日用电72MWh的 292%
    // 林业区是小组内主要的生物质供应来源
  }
}
```

### 4.4 农业区季节性产量模型（详细）

```javascript
农业区月度生物质产量(吨/天) = {
  1月: 80,     // 冬季休耕，少量残留
  2月: 80,
  3月: 90,     // 春耕准备
  4月: 100,    // 春耕
  5月: 110,    // 生长期
  6月: 130,    // 早稻区收割
  7月: 150,    // 夏收
  8月: 300,    // 秋收开始（早稻、部分玉米）
  9月: 400,    // 秋收高峰（水稻、玉米、大豆）
  10月: 350,   // 秋收后期
  11月: 150,   // 收尾、冬小麦播种

// 年总产量计算：
// (80×3 + 90 + 100 + 110 + 130 + 150 + 300 + 400 + 350 + 150 + 80) × 30 ≈ 59400 吨
```

### 4.6 生物质成分配置

**说明：** 元素分析采用收到基(ar)，即 C+H+O+N+S+M+Ash=100%

```javascript
生物质成分 = {
  工业区: {  // 生活垃圾/工业废料（质量最差），热值约10 MJ/kg
    proximate: { Moisture: 40, Volatiles: 28, FixedCarbon: 10, Ash: 22 },
    elemental: { C: 25, H: 3.5, O: 8, N: 1, S: 0.5 },
    LHV: 10.3  // 计算热值 MJ/kg
  },
  居民区: {  // 生活垃圾（略好于工业区），热值约11 MJ/kg
    proximate: { Moisture: 35, Volatiles: 35, FixedCarbon: 12, Ash: 18 },
    elemental: { C: 28, H: 3.8, O: 13.5, N: 1, S: 0.7 },
    LHV: 11.2  // 计算热值 MJ/kg
  },
  山地区: {  // 灌木枝条（中等质量），热值约15 MJ/kg
    proximate: { Moisture: 20, Volatiles: 60, FixedCarbon: 15, Ash: 5 },
    elemental: { C: 40, H: 5, O: 29, N: 0.8, S: 0.2 },
    LHV: 15.1  // 计算热值 MJ/kg
  },
  农业区: {  // 秸秆（良好质量），热值约16 MJ/kg
    proximate: { Moisture: 15, Volatiles: 65, FixedCarbon: 13, Ash: 7 },
    elemental: { C: 42, H: 5.2, O: 30, N: 0.6, S: 0.2 },
    LHV: 16.0  // 计算热值 MJ/kg
  },
  林业区: {  // 木屑/树皮（最优质），热值约18 MJ/kg
    proximate: { Moisture: 10, Volatiles: 72, FixedCarbon: 16, Ash: 2 },
    elemental: { C: 47, H: 5.6, O: 35, N: 0.3, S: 0.1 },
    LHV: 17.7  // 计算热值 MJ/kg
  }
}
```

**热值计算公式（Boie公式）：**
```
HHV = (35.16×C + 116.225×H - 11.09×O + 6.28×N + 10.465×S) / 100  (MJ/kg)
LHV = HHV - 2.442 × (M + 9×H) / 100  (MJ/kg)
```

### 4.7 小组资源平衡分析

每个小组（1个中心+4-5个成员）的综合能源结构：

**说明：**
- 风能发电潜力按装机容量 × 利用小时数估算（年利用2000-2500小时）
- 太阳能发电潜力按装机容量 × 利用小时数估算（年利用1200-1500小时）
- 各区域假设装机：风电10-30MW，光伏15-40MW（根据区域特性和建设成本调整）

| 小组中心 | 成员类型 | 生物质占比 | 风能占比 | 太阳能占比 | 总自给率 |
|---------|---------|-----------|---------|-----------|---------|
| 区域-10(工业) | 农+林+林+山 | 38% | 32% | 30% | 100% |
| 区域-12(居民) | 林+山+农+林 | 42% | 28% | 30% | 100% |
| 区域-14(居民) | 林×3+山+林 | 45% | 25% | 30% | 100% |
| 区域-23(工业) | 农+山×2+林 | 35% | 35% | 30% | 100% |
| 区域-26(工业) | 林+农+山×2 | 36% | 34% | 30% | 100% |
| 区域-28(工业) | 林×2+农+山 | 40% | 30% | 30% | 100% |
| 区域-37(工业) | 山×2+林+山+农 | 32% | 38% | 30% | 100% |
| 区域-39(居民) | 山+农+林 | 48% | 27% | 25% | 100% |
| 区域-42(居民) | 山+农×2+山+农 | 52% | 28% | 20% | 100% |
| 区域-44(居民) | 农×2+林+农 | 55% | 22% | 23% | 100% |

**特点分析：**
- **工业区为中心的小组**：生物质占比较低(32-40%)，需要更多风能补充
- **居民区为中心的小组**：生物质占比较高(42-55%)，风能需求相对较低
- **山地区成员多的小组**：风能占比高，可达35-38%
- **农业区成员多的小组**：生物质占比高，可达50-55%

## 五、电力负荷参数设计

### 5.1 负荷规模说明

假设每个区域代表一个小型城镇/园区规模：
- **工业区**：约5000户企业 + 配套设施，峰值负荷 50-80 MW
- **居民区**：约20000户居民 + 商业配套，峰值负荷 30-50 MW
- **山地区**：约2000户 + 旅游设施，峰值负荷 5-10 MW
- **农业区**：约5000户农户 + 灌溉设施，峰值负荷 10-20 MW
- **林业区**：约1000户 + 林场设施，峰值负荷 3-6 MW

### 5.2 各区域负荷特性

| 区域类型 | 日间负荷(MW) | 夜间负荷(MW) | 峰值负荷(MW) | 峰值时段 | 年用电量(GWh) |
|---------|-------------|-------------|-------------|---------|--------------|
| 工业区 | 45 | 65 | 80 | 20:00-06:00 | 480 |
| 居民区 | 35 | 20 | 50 | 18:00-22:00 | 241 |
| 山地区 | 8 | 2 | 10 | 10:00-16:00 | 44 |
| 农业区 | 12 | 8 | 20 | 灌溉季节 | 88 |
| 林业区 | 4 | 2 | 6 | 08:00-17:00 | 26 |

### 5.3 负荷曲线模型

```javascript
负荷配置 = {
  工业区: {
    dayBase: 45000,     // 白天基础负荷 kW (45 MW)
    nightBase: 65000,   // 夜间基础负荷 kW (65 MW) - 三班倒生产
    peakLoad: 80000,    // 峰值负荷 kW (80 MW)
    peakHours: [20, 21, 22, 23, 0, 1, 2, 3, 4, 5],
    variance: 8000,     // 波动范围 ±8 MW
    seasonalFactor: 1.0 // 季节系数（工业生产稳定）
    // 日用电：(45×12 + 65×12) = 1320 MWh
  },
```

## 八、数据平衡性目标

### 8.1 理想状态

每个小组应满足：
- **自给率** ≥ 80%（本组发电量/本组用电量）
- **资源利用率** ≥ 70%（实际利用/可用资源）
- **成本效益比** 相近（各组差异 < 20%）

### 8.2 平衡调节手段

1. **资源互补**：林业区提供生物质 → 工业区/居民区
2. **电力调度**：山地区白天发电 → 工业区夜间用电
3. **成本分摊**：高成本区域通过资源优势补偿
