# 测试区-1 代码求解方案（20套）

## 求解方法说明

本方案基于 `energy-system/src/ConstraintSolver.ts` 和 `energy-system/src/OptimizationEngine.ts` 中的求解算法，使用以下方法生成：

1. **参数搜索空间定义**：
   - 风电占比: [0.20, 0.22, 0.25, 0.28, 0.30, 0.33, 0.35]
   - 光伏占比: [0.55, 0.60, 0.65, 0.68, 0.70, 0.72, 0.75]
   - 生物质占比: [0.10, 0.12, 0.15, 0.18, 0.20, 0.22, 0.25]
   - 储能占比: [0.08, 0.10, 0.12, 0.15]

2. **设备选型算法**：
   - 风机: 根据目标发电量和容量因子计算数量
   - 光伏: 根据年等效利用小时数计算组件数量
   - 逆变器: 按容配比1.05-1.1选型
   - 生物质: 按年运行7000小时计算
   - 储能: 按日负荷比例配置

3. **评分函数**：
   - 约束满足度（-10分/违反项）
   - 经济性（LCOE < 0.25 +5分，> 0.35 -5分）
   - 设备匹配度（容配比1.0-1.1 +3分）
   - 储能配置（8%-12% +2分）
