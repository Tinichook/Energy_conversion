# 叶尖速比计算详解

## 一、叶尖速比公式

### 1.1 基本公式

$$\lambda = \frac{2\pi n R}{V} = \frac{\omega R}{V}$$

**参数说明：**
- λ —— 叶尖速比（无量纲）
- n —— 风机转速（r/s，转/秒）
- R —— 叶轮半径（m）
- D —— 叶轮直径（m），R = D/2
- V —— 风速（m/s）
- ω —— 角速度（rad/s），ω = 2πn

### 1.2 常用形式

**形式1（使用转速 n，单位 r/s）：**
$$\lambda = \frac{2\pi n R}{V}$$

**形式2（使用转速 n，单位 rpm）：**
$$\lambda = \frac{2\pi (n/60) R}{V} = \frac{\pi n R}{30V} = \frac{\pi n D}{60V}$$

**形式3（使用角速度 ω）：**
$$\lambda = \frac{\omega R}{V}$$

## 二、各机型的叶尖速比计算

### 2.1 计算示例：GW121/2500

**已知参数：**
- 叶轮直径：D = 121 m
- 叶轮半径：R = 60.5 m
- 额定转速：n = 13.2 rpm = 0.22 r/s
- 额定风速：V_r = 10.5 m/s

**方法1：使用 rpm**
$$\lambda = \frac{\pi n D}{60V} = \frac{\pi \times 13.2 \times 121}{60 \times 10.5} = \frac{5019.6}{630} = 7.97$$

**方法2：使用 r/s**
$$\lambda = \frac{2\pi n R}{V} = \frac{2\pi \times 0.22 \times 60.5}{10.5} = \frac{83.6}{10.5} = 7.96$$

**方法3：使用角速度**
$$\omega = 2\pi n = 2\pi \times 0.22 = 1.38 \ rad/s$$
$$\lambda = \frac{\omega R}{V} = \frac{1.38 \times 60.5}{10.5} = 7.96$$

### 2.2 所有机型在额定工况下的叶尖速比

| 型号 | D(m) | R(m) | n(rpm) | n(r/s) | Vr(m/s) | λ计算 | λ结果 |
|-----|------|------|--------|--------|---------|-------|------|
| SWT-3kW | 3.2 | 1.6 | 435 | 7.25 | 9.0 | π×435×3.2/(60×9) | **7.76** |
| SWT-10kW | 7.0 | 3.5 | 221 | 3.68 | 10.0 | π×221×7.0/(60×10) | **8.12** |
| MWT-50kW | 15 | 7.5 | 113 | 1.89 | 11.0 | π×113×15/(60×11) | **8.11** |
| GW87/1500 | 87 | 43.5 | 19.8 | 0.33 | 11.0 | π×19.8×87/(60×11) | **8.05** |
| GW121/2500 | 121 | 60.5 | 13.2 | 0.22 | 10.5 | π×13.2×121/(60×10.5) | **7.92** |

**结论：**
- 所有风机在额定工况下的叶尖速比都在 **λ = 7.76~8.12** 范围内
- 这说明所有风机都设计为在额定风速下达到**最佳叶尖速比 λ_opt ≈ 8.0**
- 不同机型的λ略有差异，但都接近最优值

## 三、不同风速下的叶尖速比变化

### 3.1 变速风机的控制策略

现代风机采用**变速控制**，在不同风速下调整转速以维持最佳λ：

**控制目标：** 在 V_in ≤ V < V_r 区间内，维持 λ ≈ 8.0

**实现方法：** 根据风速调整转速
$$n = \frac{60 \lambda_{target} V}{\pi D}$$

### 3.2 GW121/2500 在不同风速下的λ计算

**假设维持 λ_target = 8.0：**

| 风速V(m/s) | 目标λ | 所需转速n(rpm) | 实际λ | 运行模式 |
|-----------|------|---------------|-------|---------|
| 3.0 | 8.0 | 60×8.0×3.0/(π×121)=3.8 | 8.0 | 变速 |
| 5.0 | 8.0 | 60×8.0×5.0/(π×121)=6.3 | 8.0 | 变速 |
| 7.0 | 8.0 | 60×8.0×7.0/(π×121)=8.8 | 8.0 | 变速 |
| 9.0 | 8.0 | 60×8.0×9.0/(π×121)=11.3 | 8.0 | 变速 |
| 10.5 | 7.92 | 13.2（额定） | 7.92 | 额定 |
| 12.0 | 6.93 | 13.2（恒定） | 6.93 | 限功率 |
| 15.0 | 5.54 | 13.2（恒定） | 5.54 | 限功率 |

**计算示例（V=7.0 m/s）：**

**步骤1：计算所需转速**
$$n = \frac{60 \lambda V}{\pi D} = \frac{60 \times 8.0 \times 7.0}{\pi \times 121} = \frac{3360}{380.1} = 8.84 \ rpm$$

**步骤2：验证叶尖速比**
$$\lambda = \frac{\pi n D}{60V} = \frac{\pi \times 8.84 \times 121}{60 \times 7.0} = \frac{3360}{420} = 8.0 \ ✓$$

### 3.3 超过额定风速后的λ变化

当风速超过额定风速后，转速保持恒定，λ随风速增大而减小：

$$\lambda = \frac{\pi n_{rated} D}{60V}$$

**GW121/2500 示例：**

| 风速V(m/s) | 转速n(rpm) | λ计算 | λ值 | Cp |
|-----------|-----------|-------|-----|-----|
| 10.5 | 13.2 | π×13.2×121/(60×10.5) | 7.92 | 0.270 |
| 11.0 | 13.2 | π×13.2×121/(60×11.0) | 7.56 | 0.240 |
| 12.0 | 13.2 | π×13.2×121/(60×12.0) | 6.93 | 0.220 |
| 15.0 | 13.2 | π×13.2×121/(60×15.0) | 5.54 | 0.180 |
| 20.0 | 13.2 | π×13.2×121/(60×20.0) | 4.16 | 0.120 |

**结论：**
- λ 随风速增大而减小
- Cp 也随之下降
- 但功率保持在额定值（通过变桨距控制）

## 四、cp-lambda-chart.html 中的计算过程

### 4.1 HTML文件的作用

该文件**只计算 Cp-λ 的理论关系曲线**，不涉及具体风机参数。

**计算流程：**

1. **输入：** 给定一系列 λ 值（0.1 ~ 18）和 β 值（0° ~ 20°）
2. **计算：** 使用 Matlab 公式计算对应的 Cp
3. **输出：** 绘制 Cp-λ 曲线

**关键代码：**
```javascript
function calculateCp(lambda, beta) {
    const { c1, c2, c3, c4, c5, c6 } = CP_PARAMS;
    const lambdaI = 1 / (1 / (lambda + 0.08 * beta) - 0.035 / (Math.pow(beta, 3) + 1));
    const Cp = c1 * ((c2 / lambdaI) - c3 * beta - c4) * Math.exp(-c5 / lambdaI) + c6 * lambda;
    return Cp;
}
```

**注意：** 这里的 λ 是**直接给定的输入值**，不是通过 R、n、V 计算的。

### 4.2 实际应用中的计算流程

在实际风机功率计算中，流程是：

**步骤1：已知条件**
- 风速 V（来自气象数据）
- 风机参数：D、n（或转速公式）

**步骤2：计算叶尖速比**
$$\lambda = \frac{\pi n D}{60V}$$

**步骤3：计算 Cp**
- 方法A：查表2（根据风速V）
- 方法B：使用 Cp = λ/44
- 方法C：使用 Matlab 公式（根据λ和β）

**步骤4：计算功率**
$$P = \frac{1}{8} C_p \eta_1 \eta_2 \eta_3 \rho \pi D^2 V^3$$

## 五、总结

### 5.1 叶尖速比公式的通用性

**公式是通用的：**
$$\lambda = \frac{\pi n D}{60V}$$

**但每个机型的λ值不同，因为：**
1. 叶轮直径 D 不同
2. 转速 n 不同
3. 工作风速 V 不同

### 5.2 设计原则

所有风机的设计目标是：**在额定风速下达到最佳叶尖速比 λ_opt ≈ 8.0**

这样可以获得最大的风能利用系数 Cp_max ≈ 0.48

### 5.3 计算流程对比

| 场景 | 已知条件 | 计算目标 | 计算方法 |
|-----|---------|---------|---------|
| **理论分析** | λ, β | Cp | Matlab公式 |
| **风机设计** | D, Vr, λ_opt | n_rated | n = 60λV/(πD) |
| **功率计算** | D, n, V | λ → Cp → P | λ=πnD/(60V) → 查表/公式 → 功率公式 |

### 5.4 关键参数带入

**在公式 λ = 2πnR/V 中：**

- **R**：叶轮半径（m），对于 GW121/2500，R = 60.5 m
- **V**：当前风速（m/s），例如 V = 8 m/s
- **n**：当前转速（r/s），例如 n = 0.167 r/s = 10 rpm

**计算示例：**
$$\lambda = \frac{2\pi \times 0.167 \times 60.5}{8} = \frac{63.5}{8} = 7.94$$

## 六、实际应用建议

### 6.1 报告中的计算步骤

**步骤1：说明叶尖速比公式**
```
根据公式(3.7)，叶尖速比为：
λ = 2πnR/V = πnD/(60V)
```

**步骤2：计算额定工况下的λ**
```
以 GW121/2500 为例：
D = 121 m, n = 13.2 rpm, Vr = 10.5 m/s

λ = π × 13.2 × 121 / (60 × 10.5) = 7.92
```

**步骤3：查找对应的 Cp**
```
方法A：根据 λ = 7.92，使用 Cp = λ/44 = 0.180
方法B：根据 V = 10.5 m/s，查表2得 Cp ≈ 0.305
```

**步骤4：计算功率**
```
P = (1/8) × Cp × η × ρ × π × D² × V³
```

### 6.2 注意事项

1. **单位转换**：转速 rpm 转换为 r/s 需要除以 60
2. **半径与直径**：R = D/2，注意区分
3. **变速控制**：在额定风速以下，λ 保持恒定；超过后，λ 随风速减小
4. **Cp 的选择**：建议使用表2的数值（更精确）
