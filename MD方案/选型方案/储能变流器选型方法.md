# 储能变流器(PCS)选型方法

## 1. PCS概述

### 1.1 什么是PCS
储能变流器(Power Conversion System, PCS)是储能系统的核心设备，实现电池组与电网之间的双向能量转换：
- **充电模式**：将交流电转换为直流电，给电池充电
- **放电模式**：将电池的直流电转换为交流电，向电网供电

### 1.2 PCS的作用
1. **能量转换**：AC/DC双向变流
2. **功率控制**：控制充放电功率
3. **电网适配**：满足电网电压、频率要求
4. **保护功能**：过压、过流、短路保护
5. **高级功能**：黑启动、虚拟同步机、一次调频等

## 3. PCS选型原则

### 3.1 功率匹配原则
PCS额定功率应与储能系统的充放电功率需求匹配：

$$P_{PCS} \geq P_{charge/discharge}$$

其中：
- $P_{PCS}$：PCS额定功率 (kW)
- $P_{charge/discharge}$：最大充放电功率 (kW)

**推荐配置**：
- **保守配置**：$P_{PCS} = 1.0 \times P_{max}$（刚好满足）
- **标准配置**：$P_{PCS} = 1.1 \times P_{max}$（留10%裕度）
- **冗余配置**：$P_{PCS} = 1.2 \times P_{max}$（留20%裕度）

### 3.2 电池电压匹配
PCS的电池侧电压范围必须覆盖电池组的工作电压：

$$V_{bat,min} \leq V_{battery} \leq V_{bat,max}$$

**电池组电压计算**：
- 单体电池标称电压：$V_{cell}$ (如磷酸铁锂3.2V)
- 串联数量：$N_{series}$
- 电池组电压：$V_{battery} = N_{series} \times V_{cell}$

### 3.3 电流匹配
PCS的最大充放电流应满足电池组的C-rate要求：

$$I_{PCS} \geq \frac{P_{max}}{V_{battery}}$$

### 3.4 效率优先原则
- 优先选择高效率PCS，减少能量损耗
- 效率每提高1%，年损耗可减少数千度电

### 3.5 经济性原则
- 在满足技术要求的前提下，选择性价比高的设备
- 单位功率价格：元/kW

## 4. PCS选型步骤

### 步骤1：确定充放电功率需求
根据系统设计确定最大充放电功率：

**方法A：基于峰值负荷**
$$P_{PCS} = k \times P_{peak}$$
- $P_{peak}$：系统峰值负荷 (kW)
- $k$：功率系数，通常取0.5-1.0

**方法B：基于电池容量**
$$P_{PCS} = \frac{E_{battery}}{t_{discharge}}$$
- $E_{battery}$：电池容量 (kWh)
- $t_{discharge}$：期望放电时长 (h)，通常2-4小时

### 步骤2：计算电池组电压
根据电池选型结果计算电池组电压：
- 单体电池电压：查电池规格
- 串联数量：根据系统电压设计
- 电池组电压 = 单体电压 × 串联数

### 步骤3：筛选候选PCS
根据功率和电压要求筛选：
```
候选PCS = {
  PCS | P_PCS ≥ P_需求 AND 
        V_bat_min ≤ V_电池组 ≤ V_bat_max
}
```

### 步骤4：综合评分
对候选PCS进行综合评分：

$$Score = w_1 \times S_{capacity} + w_2 \times S_{efficiency} + w_3 \times S_{economy}$$

其中：
- **容量匹配度** $S_{capacity}$：
  $$S_{capacity} = 1 - \frac{|P_{PCS} - P_{需求}|}{P_{需求}}$$

- **效率得分** $S_{efficiency}$：
  $$S_{efficiency} = \frac{\eta - 90}{10}$$（归一化到0-1）

- **经济性得分** $S_{economy}$：
  $$S_{economy} = 1 - \frac{Price/P_{PCS} - Price_{min}/P_{min}}{Price_{max}/P_{max} - Price_{min}/P_{min}}$$

- **权重建议**：$w_1=0.4, w_2=0.3, w_3=0.3$

### 步骤5：确定数量
根据总功率需求确定PCS数量：

$$N_{PCS} = \lceil \frac{P_{total}}{P_{单台}} \rceil$$

**配置策略**：
- **单机配置**：选择一台大功率PCS
  - 优点：成本低、占地小
  - 缺点：无冗余
  
- **多机并联**：选择多台中小功率PCS
  - 优点：可靠性高、可扩展
  - 缺点：成本略高

## 5. 选型示例

### 示例1：小型储能系统
**需求**：
- 峰值负荷：20kW
- 电池容量：50kWh (磷酸铁锂)
- 期望放电时长：2.5小时

**步骤1：确定功率需求**
$$P_{PCS} = \frac{50kWh}{2.5h} = 20kW$$

**步骤2：计算电池电压**
- 单体电池：51.2V (16串3.2V)
- 电池组：51.2V × 10并 = 512V

**步骤3：筛选候选**
- PCS-30：30kW, 200-750V ✓
- PCS-100：100kW, 400-850V ✓（功率过大）

**步骤4：评分**
- PCS-30：容量匹配度0.67，效率95%，价格2.5万
- 综合得分：PCS-30最优

**结论**：选择 **1台 PCS-30**

## 6. PCS与电池的配套关系

### 6.1 功率配比
PCS功率与电池容量的比值称为"功率配比"或"C-rate"：

$$C-rate = \frac{P_{PCS}}{E_{battery}}$$

**典型配比**：
- **0.25C**：4小时放电（基荷储能）
- **0.5C**：2小时放电（调峰储能）
- **1.0C**：1小时放电（调频储能）
- **2.0C**：0.5小时放电（应急备用）

## 7. PCS在优化方案中的作用

### 7.1 充放电控制策略
PCS根据系统状态控制充放电：

**充电时机**：
- 新能源发电盈余时
- 电网低谷电价时段
- 负荷低于发电时

**放电时机**：
- 新能源发电不足时
- 电网高峰电价时段
- 负荷高于发电时

### 7.2 能量损耗计算
PCS充放电过程存在效率损耗：

**充电损耗**：
$$E_{loss,charge} = E_{charge} \times (1 - \eta_{PCS})$$

**放电损耗**：
$$E_{loss,discharge} = \frac{E_{discharge}}{\eta_{PCS}} - E_{discharge}$$

**往返效率**：
$$\eta_{round-trip} = \eta_{PCS,charge} \times \eta_{battery} \times \eta_{PCS,discharge}$$

典型值：
- PCS充电效率：95-98%
- 电池效率：95%
- PCS放电效率：95-98%
- 往返效率：86-91%

### 7.3 成本计算
PCS成本包括：
- **设备成本**：PCS采购价格
- **安装成本**：约为设备成本的10-15%
- **运维成本**：约为设备成本的1-2%/年

## 8. 注意事项

### 8.1 电压匹配
- 必须确保电池组电压在PCS的工作范围内
- 考虑电池充满和放空时的电压变化
- 留有10-15%的电压裕度

### 8.2 功率裕度
- 不要让PCS长期满载运行
- 建议留10-20%的功率裕度
- 考虑未来扩容需求

### 8.3 环境条件
- PCS对温度敏感，需要良好的散热
- 防护等级应满足安装环境要求
- 考虑海拔、湿度等因素

### 8.4 电网适配
- 确认电网电压等级（380V/10kV）
- 满足电网接入标准
- 考虑谐波、功率因数要求

## 10. 参考公式汇总

### 功率需求
$$P_{PCS} = \max\left(\frac{E_{battery}}{t_{discharge}}, k \times P_{peak}\right)$$

### 电池组电压
$$V_{battery} = N_{series} \times V_{cell}$$

### 电流需求
$$I_{PCS} = \frac{P_{PCS}}{V_{battery}}$$

### 数量计算
$$N_{PCS} = \lceil \frac{P_{total}}{P_{单台}} \rceil$$

### 往返效率
$$\eta_{round-trip} = \eta_{charge} \times \eta_{battery} \times \eta_{discharge}$$

### 年能量损耗
$$E_{loss,annual} = E_{throughput} \times (1 - \eta_{round-trip})$$

## 11. 总结

PCS选型的核心要点：
1. ✅ **功率匹配**：满足充放电功率需求
2. ✅ **电压匹配**：覆盖电池组工作电压范围
3. ✅ **效率优先**：选择高效率设备减少损耗
4. ✅ **经济合理**：在满足技术要求下控制成本
5. ✅ **可靠性**：考虑冗余配置提高系统可用率

**选型流程**：
需求分析 → 功率计算 → 电压计算 → 候选筛选 → 综合评分 → 确定方案
