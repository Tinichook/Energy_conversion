## 三、光伏发电功率计算

### 3.1 计算公式

#### (1) 额定功率定义（STC标准测试条件）
```
Y_PV = G_T,STC × A × η_mp,STC
```
- G_T,STC = 1 kW/m² （标准辐照度）
- A = 组件面积 (m²)
- η_mp,STC = 标准温度下转换效率 (%)

#### (2) 温度修正效率
```
η_mp = η_mp,STC × (1 + α_p × (T - 25))
```
- α_p = 功率温度系数 (%/°C)
- T = 环境温度 (°C)

#### (3) 综合降额因子
```
f = f_PV × (η_mp / η_mp,STC)
```
- f_PV = 光伏降额因子，通常取 0.95

#### (4) 实际发电功率
```
P_PV = f × Y_PV × (G_T / G_T,STC)
```
- G_T = 实际总水平辐照度 (kW/m²)

## 五、年发电量估算

### 5.1 年发电量计算公式

```
E_annual = P_installed × H_annual × η_system × (1 - D_year1) × (1 - D_annual × (n-1))
```

- P_installed = 装机容量 (kW)
- H_annual = 年等效利用小时数 (h)
- η_system = 系统效率（含逆变器、线损等），取 0.85
- D_year1 = 首年衰减率
- D_annual = 年衰减率
- n = 运行年数

## 七、计算工具

### 7.1 快速估算公式

**单块组件实际功率：**
```
P_actual = P_rated × 0.85 × (G_T / 1000)
```

**阵列日发电量：**
```
E_day = P_installed × 峰值日照小时数 × 0.85
```

**年发电量：**
```
E_year = P_installed × 1300 × 0.85 （适用于中国大部分地区）
```

## 八、注意事项

1. **实际发电功率** 受多种因素影响：
   - 辐照强度（最主要）
   - 环境温度
   - 组件表面清洁度
   - 阴影遮挡
   - 逆变器效率
   - 线路损耗

2. **降额因子** 综合考虑：
   - 温度影响：-3% ~ -7%
   - 灰尘污染：-2% ~ -5%
   - 线路损耗：-2% ~ -3%
   - 逆变器损耗：-2% ~ -3%
   - 综合降额：通常取 0.85 ~ 0.90

3. **长期衰减** 需考虑：
   - 首年衰减：1% ~ 2.5%
   - 年衰减率：0.4% ~ 0.6%
   - 25年后功率保持率：80% ~ 85%

4. **选型优先级**：
   - 效率 > 温度系数 > 衰减率 > 价格
