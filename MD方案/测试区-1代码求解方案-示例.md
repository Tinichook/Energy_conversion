# 测试区-1 优化求解方案（20套）

## 求解方法说明

本方案基于 `energy-system/src/OptimizationEngine.ts` 中的优化算法生成：

1. **8760小时仿真**：对全年8760小时进行逐时仿真，计算风光生物质发电与负荷匹配
2. **储能调度策略**：基于供需平衡的充放电控制，优化电池SOC管理
3. **多目标优化**：综合考虑可靠率、经济性、设备匹配度、系统稳定性
4. **约束满足**：确保供电可靠率≥98%，设备容量匹配合理

## 五、经济性分析

### 5.1 投资成本构成（以方案1为例）

| 设备类型 | 投资(万元) | 占比 |
|---------|-----------|------|
| 风机 | 12,000 | 32.4% |
| 光伏组件 | 11,760 | 31.8% |
| 逆变器 | 2,016 | 5.4% |
| 生物质设备 | 720 | 1.9% |
| 储能电池 | 9,520 | 25.7% |
| PCS | 1,008 | 2.7% |
| **合计** | **37,024** | **100%** |

### 5.2 LCOE计算

```
LCOE = (总投资 × (CRF + 运维率)) / 年发电量
     = (37,024万元 × 10000 × (0.08 + 0.02)) / (186,800 MWh × 1000)
     = 0.198 元/kWh
```

其中：
- CRF（资本回收系数）= 0.08（按20年寿命，8%折现率）
- 运维率 = 0.02（年运维成本占投资的2%）

### 5.3 投资回收期估算

假设上网电价 0.35 元/kWh：
- 年发电收入 = 186,800 MWh × 0.35 元/kWh × 1000 = 6,538 万元
- 年运维成本 = 37,024 × 2% = 740.5 万元
- 年净收益 = 6,538 - 740.5 = 5,797.5 万元
- 静态投资回收期 = 37,024 / 5,797.5 ≈ **6.4 年**
